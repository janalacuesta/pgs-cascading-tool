# FNLGHTC PGS Cascading Tool

A web-based interactive guide to help FNLGHTC employees understand, study, and connect with the hospital's **Performance Governance System (PGS) Journey**. Runs entirely in the browser — no installation, no login, no internet connection required after the page loads.

**Live site:** https://janalacuesta.github.io/pgs-cascading-tool/

---

## What It Contains

| Section | Description |
|---|---|
| **Overview** | Introduction to PGS at FNLGHTC, current stage, and how to use the tool |
| **Strategy Map** | Interactive accordion showing the hospital's 2028 strategic direction — Vision, Mission, Impact, Position, Core & Support Strategies, and Core Values |
| **Stages** | The four PGS stages (Initiation → Compliance → Proficiency → Institutionalization) with expandable detail cards |
| **Study Mode** | Flashcards, True or False, and Multiple Choice quiz on PGS concepts |
| **How Do I Contribute?** | A form that generates a personalized contribution statement aligned to the Strategy Map |

---

## How to Update the Site

All content is in a single file: `index.html`

1. Edit `index.html` (locally or via Claude Code)
2. Open a terminal in this folder and run:
```bash
git add index.html
git commit -m "describe your change"
git push
```
3. The live site updates automatically within ~1 minute via GitHub Pages.

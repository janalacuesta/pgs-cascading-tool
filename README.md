# FNLGHTC PGS Cascading Tool

**Far North Luzon General Hospital and Training Center**  
Performance Governance System · Cascading Tool

---

## What is this?

This is a web-based tool designed to help FNLGHTC employees understand, study, and personally connect with the hospital's **Performance Governance System (PGS) Strategy Map 2028**. It runs entirely in the browser — no installation, no login, no internet connection required after the page loads.

---

## What can you do with it?

| Section | What it does |
|---|---|
| **Overview** | Introduces the PGS, where FNLGHTC currently stands, and how to use the tool |
| **Strategy Map** | Explore the 2028 Strategy Map layer by layer — Vision, Mission, Impact, Strategic Position, Core Strategies, Support Strategies, and Core Values |
| **Stages** | Learn the four PGS stages and what each one means for FNLGHTC |
| **Study Mode** | Test yourself with 25 Flashcards, 10 True-or-False questions, and 10 Multiple Choice questions |
| **How Do I Contribute?** | Generate your personal Contribution Statement aligned to the Strategy Map |

## How to access it

### Via GitHub Pages (recommended)
The tool is live at:
```
https://<yourusername>.github.io/<repo-name>/
```

Open this link on any device — phone, tablet, or desktop — in any browser.

### Add to your Home Screen (recommended for staff)

**On iPhone / iPad:**
1. Open the link in **Safari**
2. Tap the **Share** button (box with arrow at the bottom)
3. Tap **Add to Home Screen**
4. Tap **Add**

The FNLGHTC logo will appear as an icon on your home screen. It opens like an app — no browser bar, full screen.

**On Android:**
1. Open the link in **Chrome**
2. Tap the **three-dot menu** (top right)
3. Tap **Add to Home screen**
4. Tap **Add**

## How to update the tool

1. Edit `index.html` directly in GitHub, or upload a new version
2. Rename the file to `index.html` before uploading
3. GitHub Pages will automatically redeploy within 1–2 minutes
4. Hard refresh (`Ctrl + Shift + R` / `Cmd + Shift + R`) to clear cache

---

## Technical details

- **Single file:** The entire tool is one self-contained `index.html` file
- **No dependencies:** No server, no database, no external APIs
- **No data stored:** Nothing entered in the Contribution form is saved or transmitted anywhere
- **Fonts:** Loaded from Google Fonts (requires internet on first load)
- **Compatible with:** Chrome, Safari, Firefox, Edge — mobile and desktop
